# Start Here

GitHub visitors should start in the `app/` directory.

## If you just cloned or downloaded the repo

1. Open `app/`
2. Read `README.md`
3. Run the platform-specific setup command from that folder

## Fast paths

- Windows: run `Setup.cmd`, then `Launch.cmd`
- macOS/Linux: run `bash ./setup_unix.sh`

No environment variables are required for standard local use.
